Enthät eine Hilfsklasse, die die Zahl berechnet, die einem übergebenen Buchstaben entspricht. Die Klasse wird in allen Programmteilen der Vorlage verwendet, so dass Änderungen, die Sie hier vornehmen, auch in der GUI etc. interpretiert werden. So können Sie die Abbildung von Buchstaben auf Zahlen erweitern oder verbessern.
